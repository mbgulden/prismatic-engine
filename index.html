The Agent Spectrum

Spectrum of specialized agents.
One orchestration engine.

Each agent is a wavelength in the spectrum — specialized, independent, and coordinated by the engine.

AGY
Orchestrator & Vision
Monitors output correctness and directs the swarm's workflow mapping and scheduling.
Jules
Workspace Admin & PR
Manages local repository state, runs linter checks, and audits active pull requests.
Custom
Bolt-On Agents
Custom extensions and marketplace modules plug in seamlessly with zero setup.
Hermes
Routing & Messaging
Manages websocket messaging pools, system routers, and event dispatches.
Codex
Code & Repository
Generates refactorings, populates modules, and modifies source directories.
Research
Deep Analysis
Runs concurrent search engine queries, compiles records, and extracts facts.
Pipeline

One config. Full execution.

Define your agents, set your pipeline, and Prismatic Engine handles the rest — review chains, stall recovery, and artifact handoff.

1. INPUT (YAML)

prismatic.yaml YAML
# prismatic.yaml
pipeline:
  design: agy
  build: codex
  review: jules
  vision: agy
→

2. ENGINE

Prismatic Engine parses the manifest, splits work across the agent spectrum, and manages all routing and state.

→

3. EXECUTION

AGY designs → Codex builds → Jules reviews → AGY vision-tests. Each agent checks the previous agent's work. Zero single points of failure.

Bolt-On Marketplace

Extend the spectrum.

Community-built modules that plug directly into your engine. One command to install. Zero configuration.

DATABASE

Database Connector

Run SQL agents safely inside sandboxed environments. Postgres, MySQL, SQLite.

$ prismatic install db-postgres
NOTIFY

Discord Notify

Route agent completions, alerts, and summaries directly to Discord channels.

$ prismatic install notify-discord
RUNNER

Sandbox Executor

Dockerized code execution. Every agent runs in an isolated container.

$ prismatic install sandbox-docker
